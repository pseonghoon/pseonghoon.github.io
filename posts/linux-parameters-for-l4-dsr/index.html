L4 DSR과 Linux kernel parameter

January 1, 0001

목적

L4 load balancer를 DSR(Direct Server Return) 모드로 구성할 때 real server 역할을 하는 Linux 서버에서 arp_ignore , arp_announce kernel parameter 를 설정해야 한다. 이 parameter의 의미에 대해서 명확히 설명하는 문서를 찾기 힘들었기 때문에 여기서 정리해보려 한다.

두 parameter에 대한 공식적인 설명은 아래 문서에서 찾을 수 있다. https://www.kernel.org/doc/Documentation/networking/ip-sysctl.txt
